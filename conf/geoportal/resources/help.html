Setting Basic Selection Filters

System : Sets the active system, this controls what data is downloaded and selected on the map. Changing the selection in 'Map Layer Selection' DOES NOT change this.

Date : Start/Stop dates control the range of data shown in the map and download.

Seasons/s: Controls the selection of seasons shown on the map.

Applying Basic Selection Filters

To apply the section filters above click 'Apply Filters to Map'. Make sure to change the layer in the 'Map Layer Selection' panel to match what you have selected.

Setting Spatial Filters

For more information about drawing a polygon click the 'map help' button above the map. To enter a polygon manually paste the WKT and select the projection then click 'Render WKT Text'.

Downloading Data

To download data (1) Set Basic Selection Filters (2) Set Spatial Filters (3) Select the output format and (4) click 'Download File'. This could take a few minutes so please be patient. You will be notified when the download is finished and can check the 'Downloads' tab for the current status. You can submit any number of downloads before others have completed and can continue to interact with the interface while waiting for downloads to complete.

Layer/s: Selects layers (in addition to the radar surface) for download. Only the MAT file format supports this selection.

Output Formats

CSV Format : The CSV download option includes 'all' CReSIS data, where there is a surface present (bed can be nan). The CSV file format is self explanatory.

CSV GOOD Format : The CSV GOOD download option includes 'good' CReSIS data, where there is a bed present. The CSV file format is self explanatory.

KML Format : The KML format downloads the entire path in the selected area in KML format. Complete segments are returned (even if they go outside of the selected area).

MAT Format : The MAT format downloads the radar surface, and any additional layers from (Output Layer/s) and stores them in the MATLAB m-file format.

NetCDF Format : The NetCDF format is not currently available from the OPS.
