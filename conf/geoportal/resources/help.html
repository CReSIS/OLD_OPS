Setting Basic Selection Filters

System : Sets the active system, this controls what data is download and selected on the map.

Date : Start/Stop dates control the range of data shown in the map and download.

Seasons/s: Controls the selection of seasons shown on the map.

Applying Basic Selection Filters

To apply the section filters above click “Apply Filters to Map”. Make sure to change the layer in the “Map Layer Selection” panel to match what you’ve selected.

Setting Spatial Filters

For more information about drawing a polygon click the “map help” button above the map. To enter a polygon manually paste the WKT and select the projection then click “Render WKT Text”.

Downloading Data

To download data (1) Set Basic Selection Filters (2) Set Spatial Filters (3) Select the output format and (4) click “Download File”. This could take a few minutes so please be pations. You can still interact with the portal while you are waiting for downloads to be ready.

CSV Format : The CSV download option includes "good" CReSIS data, where there is a bed present. The CSV file format is self explnatory.

KML Format : The KML format downloads the entire path in the selected area in KML format.
